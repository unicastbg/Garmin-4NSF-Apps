# Garmin 4NSF Apps Collection

A collection of simple Windows CE / Windows Mobile 6 applications for the **Garmin 4NSF (VW Maps & More / Navigon devices)**.

This project is part of ongoing experimentation with custom apps on the 4NSF platform.

---

## 📦 Included Apps

### 🔧 PocketTools

* Displays basic system information
* Includes a **network connectivity test**
* Performs a ping to `1.1.1.1` to verify internet access
* Useful for quickly checking if your **Bluetooth hotspot is working**

---

### 🚗 Speed

* Displays your **current speed (km/h)**
* Uses **GPS data from the device**

---

### 🎮 TicTacToe

* Classic Tic-Tac-Toe game

**Modes:**

* Player vs Player
* Player vs Computer

**Features:**

* Light theme
* Dark theme

---

## 📥 Installation

1. Connect your Garmin device to your computer via USB
2. Open the device storage on your computer
3. Copy the desired app folder(s) from the archive (`Apps` directory) to:

```text
\Apps\
```

4. Safely eject the device
5. Unplug the USB cable

➡️ The device will restart automatically

6. After reboot, open the **Apps drawer** on your device
7. Launch your newly installed app
